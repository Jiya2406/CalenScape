# 📅 CalenScape – Interactive Wall Calendar

A modern, responsive, and interactive wall calendar component built using React/Next.js. This project transforms a static calendar design into a dynamic and user-friendly experience with date range selection and integrated notes.

---

## 🚀 Features

* 🖼️ **Wall Calendar Aesthetic**

  * Inspired by physical wall calendars
  * Hero image paired with monthly calendar grid

* 📆 **Date Range Selection**

  * Select start and end dates
  * Visual highlighting for:

    * Start date
    * End date
    * In-between range

* 📝 **Integrated Notes Section**

  * Add notes for:

    * Entire month
    * Specific date range

* 📱 **Fully Responsive Design**

  * Desktop: Side-by-side layout (image + calendar + notes)
  * Mobile: Stacked layout for better usability

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/calenscape.git

# Navigate into project
cd calenscape

# Install dependencies
npm install

# Run development server
npm run dev
```
